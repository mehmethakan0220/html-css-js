input elementi type attribute una bağlı olarak aşağıdaki şekillerde gözükür

 
<  input type="text"> 	Displays a single-line text input field
< input type="radio"> 	Displays a radio button (for selecting one of many choices)
< input type="checkbox"> 	Displays a checkbox (for selecting zero or more of many choices)
< input type="submit"> 	Displays a submit button (for submitting the form)
< input type="button"> 	Displays a clickable button

form elementi için action attribute i

Action attribute i, form gönderildiğinde gerçekleştirilecek eylemi tanımlar. Genellikle, kullanıcı gönder düğmesine tıkladığında form verileri sunucudaki bir dosyaya gönderilir. 
 Action un alabileceği değerler.
  _blank 	    The response is displayed in a new window or tab
  _self 	    The response is displayed in the current window
  _parent 	  The response is displayed in the parent frame
  framename 	The response is displayed in a named iframe

form elementi için method attribute i

  • method="get"
  • methot="post"
default http gönderme metodu get'tir 
  Get methodu 2048 karakter uzunluğundadır.
  Sayfaları yer imlerine eklemek için kullanışlıdır.
  

  Gönderilen veriler http isteğinin gövdesine eklenir url barda gözükmez.
  Getten farklı olarak sınırsız sayıda karakter gönderebilir.
  Form gönderimlerine yer işareti koyulmaz.
  Form verileri hassas veriler içeriyorsa post kullanılmalıdır.
